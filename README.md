# Killstreak feature for TF2 Xbox 360!

## What are features?
- It keeps tracks of your kills
- Once you reach to any kill value like 5, a text will show up on screen saying `Gamertaghere is on a KILLING SPREE! (5)`

This development is currently in the works, feel free to fork the project or help me out.

# Does this differ from the Cosmos X plugin?
No. i'm making this plugin so that the killstreak doesn't need Cosmos X alone.

# Instructions
- Put the tf2killstreak.xex in the HDD or any device. (HDD is recommended)
- Open your plugin injector program
- Inject the plugin in the tf2 main menu (IIRC, you don't want background01 on screen) ![Main](https://www.audioholics.com/gadget-reviews/orange-box-review/TF2-Menu.jpg)
- Wait until a notification pops up saying it's loaded

You're done! now host a server with the killstreak.
